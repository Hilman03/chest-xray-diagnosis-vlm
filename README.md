# Chest X-Ray Diagnosis System Using Vision-Language 
# Models in PACS

## 📌 Overview
A medical AI project designing an intelligent chest X-ray 
diagnostic system that integrates Vision-Language Models 
(VLM) with a Picture Archiving and Communication System 
(PACS) clinical workflow. The system aims to assist 
radiologists by automatically analysing X-ray images and 
generating diagnostic text using AI models.

## 🎯 Problem Statement
Radiology departments face high workloads and radiologist 
shortages globally. AI-powered diagnostic tools that 
integrate with existing hospital PACS infrastructure can 
reduce turnaround time and support clinical decision-making.

## 🏗️ System Architecture
The system integrates two core AI components:
- **MedPaLM** — Vision-Language Model for medical 
  image understanding
- **TinyLLaMA** — Lightweight Large Language Model 
  for diagnostic text generation

### Workflow:
X-Ray Image Input
↓
Vision-Language Model (MedPaLM)
↓
LLM Processing (TinyLLaMA)
↓
Diagnostic Report Generation
↓
PACS System Integration
↓
Radiologist Review
